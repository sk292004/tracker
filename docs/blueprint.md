# **App Name**: Pixel Time Tracker

## Core Features:

- Activity Logging: Quickly log new activities with start and end times using a simple input form.
- Chronological Daily Log: View a clear, chronological list of daily activities.
- Activity Categorization: Assign categories to activities for improved organization and insight.
- Entry Editing: Edit past entries to correct or add details.
- AI-Powered Suggestion Tool: As the user types the name of an activity, this tool will provide category suggestions, and incorporate the user's own past choices where possible.

## Style Guidelines:

- Primary color: Salmon-beige (#E9C7AD), matching the background in the reference image to give a retro, natural, tactile feel
- Background color: Pale sand (#F2E5D9), just slightly lighter than the primary to visually separate the content
- Accent color: Chocolate (#714B33) drawn from the button colors to provide clear call-to-action highlights.
- Body and headline font: 'PT Sans', a modern and legible sans-serif for clear presentation of all activity details.
- Pixel-art icons in the style of the provided image to represent activity categories.
- Use simple, grid-based layouts inspired by the sample image, to make data and functions clear.
- Subtle pixelated transitions to reflect retro aesthetics upon interaction.
